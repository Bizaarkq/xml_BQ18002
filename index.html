En el desarrollo de esta guía he concluido, que el XML bien formado, puede definir la estructura de los datos que se pueden publicar u obtener de un sitio web.
A su vez, el agregarle css a estos XMl los datos de muestran en pantalla de manera más atractiva al usuario, en el css no se necesita sintaxis especial para el uso de las etiquetas del XML.
Y por ultimo, la utilización de conversores de XML a JSON y viceversa hacen que ambas tecnologías puedan ser utiles en terminos de no perder información, y ser más flexible este ambito de la internet.
